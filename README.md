## Certificati GXP

Bootstrap iniziale del progetto MVP per la generazione e l'invio di certificati PDF.

## Avvio locale

Esegui:

```bash
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
